# Posts Directory

Place post JSON files here for Workstream B analysis.

Each file should contain either a single post object or an array of posts:

```json
{
  "id": "post-001",
  "text": "The full text of the LinkedIn post...",
  "author": "Jane Doe",
  "date": "2025-01-15",
  "engagement": {
    "likes": 142,
    "comments": 23,
    "reposts": 8
  }
}
```

Target: ~300 posts for meaningful schema induction.
